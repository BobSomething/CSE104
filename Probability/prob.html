How does it works?
  • Number of dices - number of dices rolled each iteration
  • Simple Range - dots on each die are from 1 to Maximal number
  • Custom Range - list of dots, separated by a comma
  • Number of iterations - adjust the number of iterations
  • Specific probabilites - set custom probabilites for each roll.
